## 5. 그룹화가 안 됐을 가능성이 있는 후보 (참고)

- **포토/짧은 제목 기사**  
  - 식별자가 짧게 나오면(예: "정윤지 버디 퍼트") 같은 날·같은 행사의 다른 매체 기사와 유사도가 낮게 나올 수 있음.  
  - **권장**: 행사/대회명이 들어간 뉴스는 `extract_event_identifier` 프롬프트에서 "대회·행사는 연도·대회명·라운드 포함"을 더 강조하거나, 그레이존에서 `are_same_event` 호출을 늘려 같은 날짜·같은 인물(정윤지 등) 포토를 묶는 방안 검토.
- **영문 제목 + 한글 식별자**  
  - 일부 테크/영문 소스는 제목이 영문, 식별자는 한글로 추출됨. 서로 다른 기사끼리 유사도가 낮게 나와 그룹화되지 않을 수 있음. 현재 데이터에서는 유사도 ≥ 0.5인 쌍이 없어 추가 병합은 하지 않음.

## 6. 요약

- 수집·저장·그룹 구조는 **누락 없이 정상 동작** 중입니다.
- **유사한데 그룹화되지 않았던 1건**(리쥬란 3R 정윤지 포토)은 **동일 그룹에 병합**했습니다.
